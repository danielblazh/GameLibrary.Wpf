# Gaming Project

**GameLibrary.Wpf** is Daniel's gaming project, aimed at creating an application for managing and tracking video games in a simple and convenient way.

In the application, users can see which games they have played, which games are waiting for them, and how much time they have invested in each game. Additionally, it is possible to display the player's level in each game and rate games using stars.

The core idea is not just personal tracking, but also adding a social and competitive element, where players can compare themselves based on play time, achievements, and levels.

The application includes several screens such as a dark-themed landing page, login and registration, a game library, a profile for each game, general statistics, and an admin management screen.

The goal is to create a modern system that centralizes the entire gaming experience in one place, helping users track their progress and turning gaming into a more organized and competitive experience.
